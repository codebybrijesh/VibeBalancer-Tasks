# VibeBalancer-Tasks
Contains all task descriptions for VibeBalancer, an IEEE Envision 2026 Project.

## Task Instructions

### Task Order

Please complete the tasks in the following order:

1. **Task 1**: Learn Git and GitHub basics.
2. **Task 2**: Learn Python programming fundamentals.
3. **Task 3**: Learn about servers, REST APIs, and FastAPI.
4. **Task 4**: Week 2 → Blocking vs Non-blocking + Concurrency (Threading).

### Notes

- Each task builds on the previous one, so it is important to follow the order strictly.
- Refer to the `Learning.md` and `Assignment.md` files in each task folder for detailed instructions.
