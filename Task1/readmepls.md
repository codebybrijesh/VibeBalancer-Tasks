Task 1 - Git & GitHub Setup
Repository Link
https://github.com/codebybrijesh/VibeBalancer-Tasks
Steps I Performed

1. Connected Git with GitHub using SSH

I generated an SSH key and added it to my GitHub account to securely connect my local machine with GitHub.
2. Forked the Repository
I forked the original repository to my GitHub account using the "Fork" option on GitHub.
Cloned the Repository
git clone git@github.com:codebybrijesh/VibeBalancer-Tasks.git
>> This command downloads the repository to my local system.
 Navigated into the Project Folder
cd VibeBalancer-Tasks
>>This moves into the project directory.
5. Created a Folder inside Task1
mkdir Task1/my-folder
 This creates a new folder inside Task1.
 6. Added a File
touch Task1/my-folder/file.txt
This creates a new file inside the folder.
7. Checked Repository Status
git status
Shows modified and untracked files.
8. Added Files to Staging Area
git add .
This stages all changes for commit.
9. Committed Changes
git commit -m "Added new folder and file in Task1"
Saves the changes with a message.
10. Pushed Changes to GitHub
git push origin main
 Uploads changes to my GitHub repository.

 Conclusion

I successfully forked, cloned, modified, and pushed changes to my GitHub repository using Git commands.
